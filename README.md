# Sınav Oluşturucu(Exam Creator)
  - *Bir websitesi üzerinden çekilecek makaleler baz alınarak sınav oluşturulabilmesini sağlacak bir uygulama çalışmasıdır.*

  ## Kullanılan Teknolojiler
  ### Back-End
  - [.NET 6](https://dotnet.microsoft.com/en-us/download/dotnet/6.0)
  - [EntityFrameworkCore](https://docs.microsoft.com/tr-tr/ef/core/)
  - [AutoMapper](https://automapper.org)
  - [FluentValidation](https://docs.fluentvalidation.net/en/latest/#)
  ### Front-End
  - [ASP.NET MVC](https://docs.microsoft.com/en-us/aspnet/core/mvc/overview?view=aspnetcore-6.0)
